# MUSIC-MINI-PLAYLIST

This playlist is made from HTML, CSS and JavaScript.Its has 4 buttons play, pause, next and previous.This playlist contain 10 songs that play one after the other, one image that i generated by AI.I also add song name and title appears on  

---

## Features

- play and pause music
- next and previous track controls
- playlist display with song title and artist name and duration
- basic responsive design

---

## Technologies used

- **HTML**             -> structure of player
- **CSS**              -> styling of the player
- **JavaScript**       -> player logic and event handling
